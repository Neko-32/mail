## 使用方法
安裝 git
```bash
sudo apt install git
```  
使用 git 複製此儲存庫
```bash
git clone https://github.com/Neko-32/PiDoorNotifier.git
```  
安裝
```bash
chmod +x ./PiDoorNotifier/install.sh && ./PiDoorNotifier/install.sh
```  
執行
```bash
22-is-very-oily
```  
若遇到 400/401 錯誤，請嘗試更換/更新瀏覽器，或使用 VSCode 的 SSH 功能連線到樹梅派內進行操作
